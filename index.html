Project Vlieland

Stichting de Noordwester was planning on excavating an old WW2 bunker complex, aiming to turn it into a museum. In order to receive funding, the bunkers had to be accessible to people in wheelchairs. This was physically impossible, so we designed a 3D recreation of 1943 Vlieland so everyone could experience walking trough the complex.



Stelling 12H was put together using Unreal engine 3. My main tasks included recreating era-specific props and buildings, creating info-points which load external information to the player UI. I also cleaned up mo-cap data, set up camera sequences and made sure all of the audio was properly working ,

Amphibious Vehicle

A german vehicle i recreated, originally planned to be used as a barrier to block the player from venturing outisde the bounds of Stelling 12H

Shortly after making this the team decided to use an already developed truck to save time and this model was scrapped

This site

My first attempt at bootstrap, after setting up the LSG webshop. That webshop was basically a raw installation of Opencart with some minor tweaks. I wanted to learn more, and without feeling like i "cheated" by using a full CMS like Wordpress or Joomla. So i decided to make a portfoloio based on a framework i heard about: 'Bootstrap' . Warehouse Simulaition

For a school assigment i had to create a warehouse simulation. A train arrives and drops off it's cargo. Then 3 robots haul the cargo and store them in storage area's. They also bring items to the train. When the train is fully loaded it departs, before another train arrives in an infinite loop



The simulation written using a server-client model. Anyone could view the simulation in their browser by typing in the URL. Server-side code was written in c#, client-side written in javascript using the THREE.JS library.
