Project Vlieland

Stichting de Noordwester was planning on excavating an old WW2 bunker complex, aiming to turn it into a museum. In order to receive funding, the bunkers had to be accessible to people in wheelchairs. This was physically impossible, so we designed a 3D recreation of 1943 Vlieland so everyone could experience walking trough the complex.



Stelling 12H was created using Unreal engine 3. My main responsibilities included recreating era-specific props and buildings and developing an information distribution system to provide players with historic context. Other tasks include recording and cleaning motion-capture data, audio implementation and camera sequences.

Warehouse simulation

For an assigment i had to create a warehouse simulation. The aim was to create a simulation where a set of robots move cargo around the warehouse. A train periodically arrives to process the cargo. Afterwards the 3 robots haul the cargo and store them in predetermined storage area's. When the train is fully loaded it departs, and the cycle begins anew.


The simulation uses a basic server-client model. Anyone could view the simulation in their browser by typing in the URL. Server-side code was written in C# using ASP.NET Framework, the client-side was made using Node.JS together with THREE.JS

Dual Departure

A co-op multiplayer puzzlegame where a robot and a human must work together to get trough a deserted factory. The game is still in development, but an early version can be played here.
